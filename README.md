# Verilog Digital Circuits

A collection of Verilog implementations covering fundamental digital design concepts, including combinational logic, finite state machines, arithmetic circuits, and pipelined architectures.

## Features

- Arithmetic Logic Unit (ALU)
- Ripple-carry adders
- 8-bit adder
- Finite State Machine (FSM)
- Basic pipelined circuits
- Pipeline implementation with arithmetic and logic operations
- Modular hardware design

---

## Modules

### `alu_3.v`

Implementation of an Arithmetic Logic Unit (ALU) supporting a set of arithmetic and logical operations on binary inputs.

### `sumator.v`

Basic binary adder implementation demonstrating combinational arithmetic logic.

### `sumator8biti.v`

An 8-bit adder constructed from smaller arithmetic components, illustrating hierarchical hardware design.

### `automat.v`

Finite State Machine (FSM) implementation using sequential logic and state transitions.

### `pipeline.v`

Basic pipelined circuit demonstrating instruction/data propagation across multiple pipeline stages.

### `pipeline_add_xor.v`

Pipeline implementation combining arithmetic addition and XOR operations to illustrate pipelined combinational processing.
